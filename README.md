kwed-dl
=======

A small program to download latest tracks from remix.kwed.org.

Keeps a counter in your home-folder (\_kwedrc on windows and .kwedrc on linux).
Latest tracks are downloaded to your desktop.

I run this upon login using the Task Scheduler in windows.

There are 3 versions:

* A .NET Core version, requires .NET Core 3.1.
* A go lang version, developed using go v1.17.
* A python version, developed using python 3.6+.
